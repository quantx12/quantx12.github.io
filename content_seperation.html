CONTENT SEPERATION

As mentioned before HTML is used to structure a webpage and its contents, whereas CSS is used to present the way the contents of the webpage are displayed to the user. There are many reasons for why separating content from formatting is considered good practice and why it is important to separate the structure and the styling of a website. In the following points below, I will provide reason for why this is:

  • Manageability:
    Firstly, having a separate CSS file that controls the styling for how the contents of a webpage are displayed, helps organizations with multiple webpages manage the presentation of their website, as this affects their branding. Organizations such as Argos have multiple webpages for their various products and so they would use a common style sheet to present their webpages as it would take a lot of man hours to individually add styling to every webpage. Moreover, if the organization was undergoing rebranding or was running themes such as Christmas or Halloween then organization can easily change the styling for all their webpages by updating the single CSS file.

  • Faster:
    Secondly, having a septate stylesheet help webpages load significantly faster, this is because when as user open a website the stylesheet will automatically be saved to the browsers cache, this means that when the user loads another page from the website the Brower will use the already downloaded stylesheet to present the contents. If the styles had been integrated into the structure of the webpages, then the webpage would have to download the code to style the page every time the user goes on another page. This is especially important to users form third world countries as they might not have the most stable and fast connection to the internet as first world countries.

  • Multi-person team:
    If there are more than one person that inputs data for the content of the website then they do not need to worry about how the content they inputted is going to be presented. An example of the is a website that displays news articles, if the styling is separated and only managed by the developers the journalists do not need worry about styling the content, this would be done automatically using the stylesheet, they only need to enter the contents of their article. A separate stylesheet would ensure that the different articles produced by different journalists is presented and formatted using a unified style e.g. same font, font size and heading size.

  • Accessibility:
    Having a separated stylesheet for formatting make the website more accessible to a range of different people with different needs. This is because screen readers and braille devices would not have to deal with the formatting of the webpage and would only give the user the necessary contents of the webpage.

  • Search engine optimization (SEO):
    Websites that utilize the principle of content and style separation are more likely to perform better in search engines rankings as the search engine does not have to go through irrelevant styling code and can get a strong representation of the website based on its contents.

  • Compatibility:
    Finally, content and formatting separation helps the website be more compatible with older browsers that might not support or have disabled certain CSS styles as the user will still be able to view the HTML contents.
