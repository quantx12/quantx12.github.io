Introduction to Web development

The three-layer model of web design is the standard that is used to design websites, it refers to the three main layers of web development the structure, styles and the behavior. The three layers are commonly represented by HTML (Hyper Text Markup Language) for structure, CSS (Cascading Style Sheets) for styles and JS (Java Script) for behavior. On this website we will mainly focus on HTML and CSS. We will explain the purpose of separating content from formatting, we will demonstrate the use of different CSS selectors and we will show the use of hierarchical styles. A skeleton or a person in clothing is commonly used as analogy to explain the purpose of HTML & CSS. This is because the body of the skeleton or a person represents HTML as it the content of the website, the body is very plain and has no style, however the person/skeleton can put on clothes to stand out. In the same way CSS is used to add styling attributes to make a webpage more attractive and better tailored for the brand.

Hyper Text Markup Languages

HTML allows web developers to create the structure of a website where the content is displayed. The content of the webpage is structured using elements, an element is made up of tags such as <head> or <body> and they contain the content within the tags. Most elements have an opening such as <title> and a closing such as </title>. There are some elements referred to as void elements that donâ€™t need to be closed such as <hr>.A HTML file ends in a .html file extension.

Cascading Styles Sheets

CSS is an external file that allows web developers to specify how HTML elements are going to be displayed on a website. However, CSS can also be applied to the html file directly which is know as INTERNAL CSS and INLINE CSS. A CSS file ends with a .CSS file extension. A single CSS element consists of a CSS selector followed by the declaration contained in curly brackets. An example of this is would be: h1 { colour:red;}

Java Script

Java script is the behavior layer of the website, it is a scripting language that helps to make websites more interactive. This is because on web browsers that support JS, you are able to create content that dynamically updates, animate images and also modify the styling applied to certain elements when an action is performed by the user e.g. changing color of menu bar when user scrolls. Moreover, JS can perform calculations and validate data that has been generated by a user. An example of this is would be:
function myFunction(p1, p2) {
return p1 * p2;
}
